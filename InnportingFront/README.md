## Instrucciones de ejecución

# Backend:
Para el backend se deberá ejecutar el archivo "InnportingJava", ubicado en el package innporting que se encuentra en src/main/java.
Se utiliza el puerto 8080.
La versión utilizada es la 17.

# Frontend:
Para el front se deberá ejecutar mediante el comando "ng serve".
Se recomienda instalar las dependencias utilizando "npm install"
Se utiliza el puerto 4200.
La versión utilizada es 19.2.19
